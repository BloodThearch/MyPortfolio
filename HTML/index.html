Women Security Mobile Application

Not every smartphone comes with a built in program to help the user contact their friends in case of an emergency using just one button.

This application allows the user to send their location coordinates to a registered number via SMS, with one tap. One can also check their location and call the helpline number when they need it with one tap.

VR Gliding Simulator Game

This is a Virtual Reality Gliding Simulator Game, made in Unity.

It allows people with aerophobia to expose themselves to their fears while being in the safety of their homes. The game is also very challenging because of multiple levels.

The models were made in blender and then exported for unity in .fbx format.
